Controle de Tarefas

O Controle de Tarefas é uma aplicação desenvolvida em Java com integração a banco de dados, focada na organização e gerenciamento eficiente de atividades do dia a dia.

O sistema oferece uma interface simples e funcional, permitindo ao usuário gerenciar suas tarefas de forma intuitiva. Entre as principais funcionalidades, destacam-se:

➕ Cadastro de novas tarefas
📋 Visualização das tarefas em uma tabela integrada ao sistema
✏️ Edição de tarefas existentes
❌ Remoção de tarefas

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento com Java, manipulação de dados e operações CRUD (Create, Read, Update, Delete), além de demonstrar boas práticas na construção de aplicações com persistência de dados.
